Domain Name Server(DNS)

What is DNS?

DNS (Domain Name System) is a system that converts domain names like www.google.com into IP addresses that computers can understand.

Example: www.google.com → 142.250.190.78

Importance of DNS

  • Makes it easy to access websites
  • No need to remember IP addresses
  • Helps connect users to the correct server

How DNS Works

  1. User types a domain name
  2. DNS finds the IP address
  3. IP address is returned to the browser
  4. Website loads

Types of DNS Servers

  • Recursive Resolver
  • Root Server
  • TLD Server
  • Authoritative Server

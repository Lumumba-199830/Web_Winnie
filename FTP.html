File Transfer Protocal

File Transfer Protocol (FTP) is a method used to transfer files between computers over a network such as the internet. It allows users to upload and download files from a server.

Functions of FTP

  • Uploads files to a server
  • Downloads files from a server
  • Manages files (delete, rename, move)
  • Transfers large files easily

How FTP Works

  1. User connects to an FTP server
  2. User logs in
  3. Files are transferred
  4. Connection is closed

Examples

  • FileZilla
  • WinSCP
  • Cyberduck

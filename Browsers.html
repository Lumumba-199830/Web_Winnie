Web Browsers

A web browser is a software application that allows users to access, search, and view content on the World Wide Web by communicating with web servers. When you type a URL or click a link, the browser sends a request over HTTP or HTTPS, fetches files such as HTML, CSS, and JavaScript, and then renders them into the text, images, and interactive pages you see on screen. Modern browsers like Chrome, Firefox, Safari, and Edge also provide features such as tabs, bookmarks, history, and security tools to make navigating the internet faster, safer, and more convenient

  1. It sends HTTP requests to server
  2. Receives HTML/CSS/Java Script responses
  3. Renders them to visual pages

World Wide Web(WWW)

WWW is a system of interconnected webpages and information that aims to give universal access to a large array of documents.

How it is related to IT.

  1. Data Transmission on Network

    2. The web depends on IT networking hardware to deliver text images audio and video globally.

  2. Scale and IT management

    3. The web requires massive IT infrastructure including data centers, cloud platforms and context delivery networks all managed by IT professionals.

  3. FFoundation of modern computing infrastructure

    4. The web runs on the client-server model where a client requests information and a server delivers it.

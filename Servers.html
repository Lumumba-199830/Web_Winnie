Web Server

What is a Web Server?

A web server is a computer system or software that stores, processes, and delivers web pages to users when requested through a browser.

Example: www.google.com sends pages through a web server.

Functions

  • Stores web pages
  • Handles requests
  • Delivers files
  • Supports many users

How It Works

  1. User types URL
  2. Browser sends request
  3. Server processes request
  4. Page is returned

Hypertext Transfer Protocol

What is HTTP?

HTTP (HyperText Transfer Protocol) is a communication protocol used to transfer data between a web browser and a web server. It is the foundation of communication on the World Wide Web.

How HTTP Works

  1. User enters a website URL
  2. Browser sends an HTTP request to the server
  3. Server processes the request
  4. Server sends back a response (web page)
  5. Browser displays the page
>

Features of HTTP

  • Stateless protocol
  • Request-response model
  • Used for web communication
  • Transfers web pages and files
